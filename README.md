# Arrays
<h3>
Java array program from TakeYouForward. </h3>

## Main Branch
- Largest Number in Array

## Branches
- is-sorted : Checks if array is sorted in Increasing or Decreasing order
- left-rotate : Shifts the array one element to the left
- linear-search : Searches the array for a number
- max-consecutive-ones : Finds streak of number "1"
- missing-number : Finds the missing number in (n-1) array with elements range [1,n]
- remove-dupes-sorted : Removes duplicates and returns number of unique elements
- rotate-by-k : Rotate the array left or right by "k" places
- second-smallest-largest : Returns Second-Largest and Second-Smallest number
- zeroes-to-end : Shifts all the '0's to end of array
